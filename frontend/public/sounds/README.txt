# Sound Files for Alert Notifications

This directory contains audio files used for alert notifications.

## Required Files

Add the following MP3 audio files to this directory:

1. **default.mp3** - Default alert sound
2. **chime.mp3** - Gentle chime sound
3. **bell.mp3** - Bell ringing sound
4. **alert.mp3** - Urgent alert sound

## Sources

You can use free sound effects from:
- [Freesound.org](https://freesound.org/)
- [Zapsplat.com](https://www.zapsplat.com/)
- [Pixabay Audio Effects](https://pixabay.com/sound-effects/)

## Specifications

- Format: MP3
- Duration: 1-3 seconds recommended
- Volume: Normalized to -3dB
- Sample rate: 44.1kHz or 48kHz

## Temporary Placeholders

If you don't have sound files yet, the system will handle errors gracefully.
Alerts will still show visual notifications and browser notifications (if permitted).
